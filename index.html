Here is a list of official and generated SDKs and Tests that can be used to connect to the BitScoop Platform.

Offical BitScoop SDKs

BitScoop JavaScript Node SDK

This is the official node.js library for calling the BitScoop API.

Supports Node 6.0 and up.

Create sofware using any Node package or framework. Works with code as a service platforms like AWS Lambda, Azure Cloud Functions, IBM’s OpenWhisk, and Google Cloud Functions.

Coming Soon: BitScoop Python SDK

This is the official Python library for calling the BitScoop API.

Supports Python 2 and 3.

Code using any Python package. Works with code as a service platforms like AWS Lambda. Pairs well with Python data science libraries like Jupyter notebook, TensorFlow, Pandas, Numba, and Matplotlib.

Generated Swagger SDKs

You can generate an SDK in many languages for the BitScoop Platform using a variety of 3rd party services.

First download the Swagger specification files for the API Provider Maps you wish to use. Then import these Swagger specification files into the SDK generation tool of your choice.

APIMATIC Platform SDKs

C#, Python, Java, Node, PHP, Ruby, Android, iOS

APIMATIC is a premium service that can generate customizable SDKs from a variety of API description standards. APIMATIC also includes CI / CD Tools Integration, Git Deployment, Test Case Generation, and Generated Docs.

Use APIMATIC DX Kits to consolidate millions of API endpoints into simple SDKs with APIMATIC’s Code-Gen As A Service.

Need to import or export an API Description in a format other than Swagger 2.0? Use APIMATIC Transformer.

Supports API Blueprint Swagger (1.0 - 3.0, JSON/YAML), WADL, WSDL, Google Discovery, RAML 0.8 - 1.0, Mashery I/O Docs, HAR 1.2 Postman Collection 1.0 - 2.0 and Mashape

Swagger SDK Codegen

C#, C++, Python, Java, JavaScript, PHP, Ruby, Go, Android, iOS, Swift, Scala, Perl, BASH, and more.

Swagger Codegen is an Open Source project and Public API that can generate customizable SDKs from the Swagger 2.0 API description standard. Swagger Codegen also includes Test Case Generation and Generated Docs.

SDKs are generated through their API endpoint or using a command line utility. SDK Languages are supported through a variety of open source templates.

Testing and Monitoring

Test and Monitor using Postman

Test and Monitor any API through the BitScoop Platform or directly using Postman. Postman Collections are test cases for API endpoins and calls. The BitScoop Platform automatically generates Postman Collections for any API Map. Postman Monitors allow for scheduling Postman Collection Tests to run on an automated schedule with team alerts.

Postman lets you generate snippets of code in various languages and frameworks including cURL, Python, Java, JavaScript, PHP, Ruby, Go, Android, iOS, Swift, OCaml, Perl, BASH, and more.
